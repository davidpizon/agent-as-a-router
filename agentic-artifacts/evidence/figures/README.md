# Figure And Table Evidence

This open-source bundle prioritizes executable tables and checked-in matrices.
Rendered paper figures are not required for reproduction.

Useful table/data products:

- `evidence/tables/acrouter_release_summary.csv`: compact ID/OOD ACRouter
  summary.
- `evidence/tables/ood176_baseline_table.md`: paper-style OOD176 baseline table.
- `evidence/tables/ood176_baseline_metrics.csv`: machine-readable OOD176
  baseline metrics.
- `outputs/current/summary.md`: checked-in ID + legacy OOD112 summary.
- `outputs/baselines_ood176/baseline_table.md`: canonical baseline table.
- `data/matrices/phase2_ood/unified/results_long.csv`: normalized OOD176
  result matrix.
